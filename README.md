### Adds the "Swift Flight" Enchantment

Each level increases the flight speed of happy ghasts
- Swift Flight I: 40% faster
- Swift Flight II: 60% faster
- Swift Flight III: 80% faster

**How to obtain:** 
- Has a 8% chance to find it in end city chests

**How to use:**
- Can be put on happy ghast harnesses
